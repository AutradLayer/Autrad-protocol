# AUTRAD

> The CME for Machine Resources.

Autrad is the financial infrastructure for AI compute markets on Solana. Trade, hedge, and secure machine resources with programmable derivatives designed for inference, GPU capacity, and AI workloads.

Instead of treating compute as a static service, Autrad transforms machine resources into financial instruments that can be priced, reserved, hedged, and settled entirely on-chain.

---

# 🚀 What is Autrad?

AI compute has become one of the world's fastest-growing commodities.

Yet pricing remains fragmented, volatile, and difficult to manage across GPU providers, inference networks, and decentralized compute marketplaces.

Autrad introduces a financial marketplace where compute becomes a tradable asset.

Developers can lock future inference prices.

Infrastructure providers can hedge future revenue.

Liquidity providers earn yield from settlement activity.

Everything settles transparently on Solana.
